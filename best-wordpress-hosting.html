Responsive Navigation Menu

This is a demonstration of the responsive navigation menu. Resize your browser window to see how the menu changes between desktop, tablet, and mobile views.

On mobile devices (width ≤ 768px), you'll see a hamburger menu icon. On tablet and desktop devices (width > 768px), you'll see the full horizontal menu.

How it works:

  • Desktop/Tablet (≥769px): Full menu displayed horizontally
  • Mobile (≤768px): Hamburger icon shown, menu hidden
  • Click hamburger to toggle mobile menu
  • Menu closes automatically when a link is clicked (mobile)
  • Smooth animations for opening/closing
